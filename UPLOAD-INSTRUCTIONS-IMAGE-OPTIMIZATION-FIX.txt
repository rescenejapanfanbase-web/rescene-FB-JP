RESCENE JAPAN FANBASE 画像最適化エラー修正の反映手順
====================================================

1. このZIPを展開します。
2. 展開した中身をすべてGitHubリポジトリ直下へアップロードします。
   ZIPファイル自体や、update-packageフォルダをそのまま置かないでください。
3. 同名ファイルは上書きし、「Commit directly to the main branch」で保存します。
4. GitHubの Actions → Sync All Site Content → Run workflow を実行します。
5. 画像最適化欄が次のように表示され、最後まで完了すれば成功です。

   最適化不要: 7件 / 失敗: 0件

初回実行時は、リポジトリ直下へ誤配置されていた重複WebPも自動整理されます。
assets/optimized内と完全一致するファイルだけが削除対象なので、元画像は削除されません。
