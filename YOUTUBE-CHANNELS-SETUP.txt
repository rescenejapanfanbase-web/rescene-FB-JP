RESCENE / ウォニ YouTube全公開動画同期 反映手順

【取得対象】
- RESCENE Official（@RESCENE_official）
- 안녕하세요원이입니다잘부탁드립니다（@helloiamwoninicetomeetyou）
- 各チャンネルの「動画」「ショート」「ライブ」タブに表示される公開動画すべて

【仕組み】
- GitHub Actionsでyt-dlpを自動インストール
- 各タブをページ送りを含めて全件取得
- ライブ候補は動画ごとの詳細情報も確認
- ライブは推定投稿日ではなく実際の配信開始日時を優先
- 同じ動画が別IDや複数タブから取得された場合は正規IDへ統合
- 確認済みの終了ライブは次回以降再利用して同期を軽量化
- data/youtube-channels.jsonへ保存
- 3時間ごとに自動同期
- APIキーやGitHub Secretsの追加は不要

【初回反映】
1. 更新ZIPを展開します。
2. フォルダ構成を保ったままGitHubリポジトリ直下へ上書きします。
3. Commit directly to the main branchでCommitします。
4. Actions → Sync YouTube Channels → Run workflowを実行します。
5. 初回は過去動画を全件取得するため、通常の更新より処理時間が長くなる場合があります。
6. Actionsが成功したらyoutube.htmlをCtrl+F5で再読み込みします。

【ページ表示】
- データは全件保存します。
- 画面は動作を軽くするため最初に24件を表示します。
- 「さらに24件表示」で続きを表示できます。
- チャンネル、ショート、通常動画、ライブで絞り込めます。
- 動画タイトル検索を利用できます。

【対象外】
- 非公開動画
- 削除済み動画
- チャンネルの公開タブに表示されない限定公開動画
