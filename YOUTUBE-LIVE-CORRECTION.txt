YouTubeライブ重複・日時修正
更新日: 2026-07-19

【修正内容】
- ライブタブの動画を、動画ごとの詳細メタデータで再確認
- 推定投稿日ではなく、ライブの実際の配信開始日時を優先
- 一度保存された誤った推定日時が残り続ける問題を修正
- 同じ動画が別ID経由で取得された場合、YouTubeの正規IDへ統合
- 動画・ショート・ライブの複数タブに出る同一動画を1件へ統合
- ライブではない項目は通常動画またはショートへ戻す
- 終了済みライブの確認結果を保存し、次回以降の同期を軽量化
- 配信中・配信予定・処理中のライブだけは次回も再確認

【反映後】
1. GitHub Actionsを開きます。
2. Sync YouTube Channelsを選びます。
3. Run workflowを実行します。
4. 初回は過去ライブの詳細確認があるため、通常より長くなる場合があります。
5. 成功後、youtube.htmlをCtrl+F5で再読み込みします。

【重要】
- data/youtube-channels.jsonは更新ZIPに含めていません。
- 現在GitHubに保存されている全動画データを残したまま、Actionsで修正後のデータへ更新します。
