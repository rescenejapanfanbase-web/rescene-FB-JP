RESCENE JAPAN FANBASE 同期状況ページ

ページ:
- sync-status.html

確認できる内容:
- Notionスケジュール同期の最新実行結果、公開件数、データ変更日時
- Notionニュース同期の最新実行結果、公開件数、Notion取得件数、データ変更日時
- YouTube同期の最新実行結果、チャンネル数、全公開動画件数、通常動画・ショート・ライブの種類別件数、データ変更日時
- トップページ用ガイド更新情報の生成結果、件数、データ変更日時
- 自動バックアップの最新実行結果とArtifactへのリンク
- サイト内リンク・画像チェックの最新実行結果
- 各GitHub Actionsの手動実行先と最新ログ

仕組み:
- 公開件数とデータ変更日時は data/*.json から取得
- 成功・失敗・実行中と最終実行日時はGitHub REST APIの公開情報から取得
- GitHub APIの呼び出しを減らすため、通常表示では5分間キャッシュ
- 「再読み込み」ボタンではキャッシュを使わず最新状況を取得

注意:
- 静的サイトのため、このページ自体にログイン機能や秘密情報はありません。
- GitHub Actionsの手動実行にはGitHubへのログインとリポジトリ権限が必要です。
- データに変更がない正常同期では data/*.json の generatedAt は変わりません。
  実際の最終同期時刻はGitHub Actionsの「最終同期実行」を確認してください。
