
RESCENE JAPAN FB - 第1弾
========================

画像ファイルを一切使わず、HTML / CSS / JavaScriptだけで制作した静的サイトです。
GitHub Pagesへそのままアップロードできます。

主なページ
- index.html: トップページ
- about.html: RESCENE紹介
- members.html: メンバー
- discography.html: 作品一覧
- news.html: ニュース（絞り込み対応）
- schedule.html: スケジュール
- mv.html: 公式YouTubeへの入口
- streaming.html: ストリーミング基本ガイド
- voting.html: 投票ガイド土台
- links.html: 公式リンク
- contact.html: 問い合わせ案内

更新しやすい場所
- ニュース: js/data.js
- 全体デザイン: css/common.css の :root
- ファンベースの連絡先: contact.html

注意
- この版に画像・画像プレースホルダー・背景画像はありません。
- CSSの円形装飾とグラデーションなしの色面だけで構成しています。
- 最新情報やイベント詳細は、公開前に公式発表と照合してください。
